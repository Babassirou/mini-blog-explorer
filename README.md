PROJECT TITLE: MINI BLOG EXPLORER 

What does project does? 

This project is a mini blog explorer. It is built with React and Public API (https://jsonplaceholder.typicode.com/posts) which is an array of objects. Each object have the follwing keys: userId, id, title and a body.

the project fetch the api through App.jsx uses multiple states management to store and update the state of the data that is display.

The react concept used are States which help us to manipulate the data and update it. useEffect, allowed us to fetch external API while embeding error handling logics. Conditional rendering and  error handling.

To locally you will need to run npm run dev inyour terminal and launch the localhost. 

The biggest challenged I faced was wrapping my head around how the component  the states. Also how two component like PostCard and PostGrid passed logic between one another to display individual cards. This was a turning point for me. 

I have learned in a more in-depth approach how to take data from outside source manipulate and beutifully display it in UI.
